Desafio Técnico de People Analytics: Sourcing & AI Strategy

Este repositório contém uma solução avançada para o desafio de People Analytics, focada em otimizar o recrutamento através de dados e Inteligência Artificial.

🚀 Estrutura da Solução

1.
Análise de Funil: Mapeamento de taxas de conversão e identificação de gargalos.

2.
Modelagem Preditiva (IA): Uso de Random Forest para prever contratações e identificar sinais de sucesso.

3.
Análise de Persistência: Definição do ponto de inflexão para follow-ups (quando parar de insistir).

4.
Insights de Negócio: Respostas detalhadas para as perguntas estratégicas do desafio.

📂 Arquivos

•
relatorio_final_v2.pdf: Relatório executivo completo.

•
people_analytics_notebook.ipynb: Notebook com todo o código e visualizações.

•
analysis.py: Scripts de processamento de dados.

•
sourcing_data.csv: Base de dados utilizada.

🧠 Metodologia de IA

Utilizamos Machine Learning para transformar dados brutos em decisões. O modelo identificou que Anos de Experiência e Score Comportamental são os maiores preditores de sucesso, permitindo que recrutadores foquem nos candidatos com maior potencial de fechamento.
